# rAIdio.bot

**AI music production studio for Windows.**  
Generate music, clone voices, extract stems, build playlists, 
and run your own personal radio station — all locally, all on your hardware.

---

## Filing an issue

Use the links below or click **New Issue** to choose a template.

- 🐛 **[Report a Bug](https://github.com/memescreamer/rAIdio.bot/issues/new?template=bug_report.md&labels=bug,user-reported)** — something is not working correctly
- 💥 **[Report a Crash](https://github.com/memescreamer/rAIdio.bot/issues/new?template=crash_report.md&labels=crash,user-reported)** — the app crashed or became unresponsive
- 💡 **[Request a Feature](https://github.com/memescreamer/rAIdio.bot/issues/new?template=feature_request.md&labels=feature-request)** — suggest something new or an improvement

### What makes a good bug report

The faster we can reproduce it, the faster it gets fixed. The most useful reports include:

- Your GPU model and VRAM
- Your Windows version
- Which tab or operation triggered the issue
- Whether it happens every time or intermittently
- The ComfyUI console output if the app crashed
